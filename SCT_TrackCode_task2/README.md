# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Overview
This project applies **K-Means Clustering** to group retail customers based on their purchasing behavior.

It helps in identifying distinct customer segments for targeted marketing strategies.

---

## 🎯 Objective
To segment customers into clusters based on features such as:
- Annual Income
- Spending Score

---

## 🧠 Model Used
- K-Means Clustering (Unsupervised Learning)

---

## 📊 Dataset Requirements
Dataset should include:
- `Annual Income`
- `Spending Score`

---

## ⚙️ Workflow
1. Load dataset  
2. Select relevant features  
3. Apply K-Means clustering  
4. Assign cluster labels  
5. Visualize clusters using Matplotlib  

---

```bash
pip install pandas scikit-learn matplotlib
